# trading-simulator
## 项目目标
基于SpringBoot实现证券交易撮合模拟系统，核心完成「订单校验→对敲风控→撮合匹配→交易回报」全链路，支持行情、撤单、数据分析等可插拔增强模块。

## 核心功能列表
### 基础目标（必选）
1. 交易转发：接收JSON订单，完成基础合法性校验，输出确认/非法回报
2. 对敲风控：检测同一股东号自买自卖行为，拦截违规订单并输出非法回报
3. 模拟撮合：维护订单簿，实现价格优先+时间优先撮合，支持零股成交和成交价生成
4. 交易回报：统一输出JSON格式的确认/成交/非法回报

### 高级目标（可选）
1. 行情接入：解析行情JSON，撮合时校验买卖价格与行情一致性
2. 撤单支持：处理撤单请求，输出撤单确认/非法回报
3. 性能优化：分析瓶颈并优化吞吐量/延时
4. 数据分析：存储交易历史并输出分析结果
